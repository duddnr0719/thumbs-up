## 🛠️ 사용 가능한 도구

### 1️⃣ **번들 시각화** (Rollup Plugin Visualizer)

#### 설정

- ✅ `vite.config.ts`에 이미 설정됨
- ✅ 빌드 시 `dist/stats.html` 자동 생성

#### 사용 방법

```bash
# 1. 프로덕션 빌드 실행
npm run build

# 2. 브라우저에서 열기
open dist/stats.html

# 또는 파일 경로로 직접 열기
file:///path/to/frontend/dist/stats.html
```

#### 분석 항목

- 📊 각 번들 크기 (Rendered, Gzip, Brotli)
- 📦 의존성 구조 시각화
- 🔍 최적화 대상 식별

#### 현재 번들 상태

```
Total (gzip): 66.60 KB
├── React ecosystem: ~49 KB (74%)
├── App code: ~14 KB (21%)
└── Styles: ~2.5 KB (4%)
```

### 2️⃣ **Lighthouse** (종합 성능 평가)

#### 설치

```bash
npm install -g lighthouse
# 또는
npx lighthouse
```

#### 실행 방법

```bash
# 기본 실행 (Chrome 자동 시작)
npx lighthouse http://localhost:5175

# JSON 리포트 생성
npx lighthouse http://localhost:5175 --output=json --output-path=report.json

# HTML 리포트 생성
npx lighthouse http://localhost:5175 --output=html --output-path=report.html

# 여러 형식 동시 생성
npx lighthouse http://localhost:5175 \
  --output=json \
  --output=html \
  --output-path=./lighthouse-report
```

#### 핵심 지표 (Core Web Vitals)

| 약자 | 의미                      | 목표     | 현재       |
| ---- | ------------------------- | -------- | ---------- |
| FCP  | First Contentful Paint    | < 1.8 s  | 5.0 s      |
| LCP  | Largest Contentful Paint  | < 2.5 s  | 8.8 s      |
| CLS  | Cumulative Layout Shift   | < 0.1    | 0.0 ✅     |
| INP  | Interaction to Next Paint | < 200 ms | ~16 ms ✅  |
| TTFB | Time to First Byte        | < 600 ms | ~100 ms ✅ |

### 3️⃣ **Chrome DevTools** (실시간 성능 분석)

#### Performance 탭

**사용 방법:**

1. 브라우저에서 `http://localhost:5175` 열기
2. F12 또는 우클릭 → "검사" 클릭
3. "Performance" 탭 클릭
4. ⭕ 버튼 클릭하여 기록 시작
5. 퀴즈 진행 (1~2개 질문, 페이지 전환)
6. 다시 ⭕ 버튼 클릭하여 기록 중지

**분석 항목:**

- 📊 **FPS 그래프**: 프레임 레이트 (목표: 60 FPS)
- ⏱️ **Main 스레드**: 렌더링 시간 분석
- 🧠 **메모리**: 힙 사용량 추이
- 🎨 **렌더링**: 페인트 영역 시각화

#### Lighthouse 탭 (DevTools 내장)

**사용 방법:**

1. DevTools 열기 (F12)
2. "Lighthouse" 또는 "PageSpeed Insights" 탭
3. "분석 시작" 클릭
4. 자동 분석 완료 (1~2분)

**결과 해석:**

- 🟢 90-100: 우수
- 🟡 50-89: 개선 필요
- 🔴 0-49: 심각한 개선 필요

### 4️⃣ **React DevTools Profiler** (컴포넌트 성능)

#### 설치

- React DevTools 브라우저 확장 설치
- [Chrome](https://chrome.google.com/webstore) 또는 [Firefox](https://addons.mozilla.org)

#### 사용 방법

1. DevTools 열기
2. React DevTools 탭 → "Profiler"
3. ⭕ 버튼 클릭 → 상호작용 수행 → 중지
4. 각 컴포넌트의 렌더링 시간 확인

**분석 항목:**

- 📊 컴포넌트별 렌더링 시간
- 🔄 불필요한 리렌더링 추적
- ⚡ 성능 병목 지점 식별

#### 현재 성능

```
App.tsx: ~2ms
├── QuizHeader: <1ms
├── ProgressBar: <1ms
├── QuestionCard: ~1ms
├── ProgressBar: <1ms
├── PaginationControls: <1ms
└── (또는) ResultCard: ~2ms
```
